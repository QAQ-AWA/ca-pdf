# ca-pdf

[📘 Documentation Portal](https://qaq-awa.github.io/ca-pdf/) · 支持 **中文 / English** 语言切换

ca-pdf 是一套自托管的 PDF 电子签章平台，提供内置证书颁发机构（CA）、时间戳服务和企业级审计能力。项目支持本地部署与云端落地，帮助组织在自有基础设施中完成可信的数字签章流程。

## 快速上手

```bash
git clone https://github.com/QAQ-AWA/ca-pdf.git
cd ca-pdf
cp .env.example .env
cp .env.docker.example .env.docker
./deploy.sh up
```

更多部署、开发和运维说明，请前往文档站点查看对应章节。
